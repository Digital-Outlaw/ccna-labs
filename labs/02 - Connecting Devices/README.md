# Lab 02 - Connecting Devices

## Objective
Connect network devices correctly using the appropriate cables and interfaces.

## Devices Used
- Routers
- Switches
- Firewalls
- PCs / Servers

## Tasks Completed
- Placed all devices according to the diagram
- Selected appropriate cables for each connection
- Connected PCs to switches using straight-through cables
- Connected routers to switches using straight-through cables
- Connected router to router using crossover cable
- Connected fiber ports where appropriate

## Verification / Testing
- All device interface lights are green
- Connections match the diagram

## Skills Learned
- Correct cable selection for different devices
- How to connect devices properly in Packet Tracer
