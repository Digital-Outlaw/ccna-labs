# Lab 01 - Basic Topology Setup

## Objective
Create a basic network topology using routers, switches, firewalls, PCs, and servers.  
Practice placing devices, naming them, and connecting them correctly.

## Devices Used
- Cisco 2911 Router (x2)
- Cisco 2960 Switch (x2)
- Cisco ASA 5505 Firewall (x2)
- PC (x2)
- Server (x2)
- Laptop (Attacker)

## Tasks Completed
- Placed and named all devices according to the network diagram
- Used "Automatically Choose Connection Type" to connect interfaces
- Ensured consistent naming for readability

## Skills Learned
- Understanding different network device roles
- Interface selection & cable types
- Proper network diagram organization
